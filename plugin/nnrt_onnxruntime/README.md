# nnrt_onnxruntime

StepIt plugin for neural network inference on general x86_64 CPUs.

### Provided Factories

- `stepit::NnrtApi`: `onnxruntime`

### Executables

- `onnx_metadata2yaml`: dumps custom ONNX model metadata into a YAML file.

	```shell
	onnx_metadata2yaml <model.onnx> [output.yaml]
	```
